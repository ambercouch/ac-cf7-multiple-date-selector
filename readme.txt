Adds multiple date selector to Contact Form 7.

== Description ==

A Contact Form 7 add on that allows you to create a date selector field which allows users to submit multiple dates.

== Installation ==

Use WordPress' Add New Plugin feature, searching "AC CF7 Form Field Repeater", or download the archive and:

1. Upload the plugin files to the `/wp-content/plugins/ac-cf7-multiple-date-selector` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Edit a contact form or create a new contact form with Contact Form 7
4. Use the AC Multi Date form tag generator button in the Contact Form 7 editor to add the `[acmultidate acmultidater-xxx]` tags to your contact form.
5. Add the AC Multi Date tags to your email template.s

This plugin requires the Contact Form 7 plugin to be installed and activated.
